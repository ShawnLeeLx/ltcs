# C10 经济平衡实测 — 2026-06-16

## 目标
调整游戏经济系统，确保：
1. 前期(Ch1-3)够买必需品，不卡关
2. 中期(Ch4-8)收支紧张但不缺金
3. 后期(Ch9-12)有足够金币回收机制

## 问题诊断
原始经济收支比0.64（持续亏损），主要问题：
- **Ch1买不起铁剑+皮甲**（142g < 160g）
- **强化1.5^level太陡**（传奇+5需39562g）
- **战斗不掉金币**导致早期缺金
- **后期金币溢出**（无回收机制）

## 调整方案（6项）

| 调整 | 原值 | 新值 | 影响 |
|------|------|------|------|
| 强化指数 | 1.5^level | 1.3^level | 传奇+5从39562→27129g，+10从339987→127855g |
| 铁剑/皮甲 | 80g/80g | 60g/60g | Ch1购入门槛从160→120g |
| 霜刃 | 200g | 160g | Ch2购入更容易 |
| 战斗金币 | 0 | 5-10+等级*0.5/敌人 | 早期每战~10g，后期~50g |
| 出售价 | 40% | 50% | 卖装备收益提升25% |
| Ch1故事金币 | 20/35/50 | 40/55/75 | 新手引导期更宽裕 |
| 训练场费用 | 线性(50+20n) | 指数(50*1.10^n) | 30次训练消耗9K金币 |
| 龙祭系统 | 无 | 6种永久属性提升(Lv30+) | 全满246K金币回收 |

# V15 开发完成 — 故事选择 + 连锁任务

## 目标
P0任务：实现故事选择系统（剧情分支）和连锁任务系统（任务链），提升故事50%→65%、任务35%→50%

## 完成内容

### 1. 故事选择系统 (STORY_CHOICES)
- **5个关键选择点**: ch1_arrival(3选1)、ch2_forest(2选1)、ch4_mist(3选1)、ch7_dragon(3选1)、ch11_abyss(3选1)
- **选择效果接入游戏核心**:
  - `combat_bonus`: +2体魄(永久)
  - `scout_bonus`: +2敏捷(永久)
  - `social_bonus`: +2魅力(永久) + 好感度提升+10%
  - `atk_bonus`: +3体魄(永久)，敌人更强
  - `int_bonus`: +3智力(永久)，解锁魔法对话
  - `luk_bonus`: +3幸运(永久)，宝箱掉率+15%(接入generateRandomDrop)
  - `dragon_ally`: 龙族好感大幅提升
  - `human_ally`: 人类阵营好感提升，铁匠铺额外折扣+5%(接入getNpcDiscount)
  - `neutral_path`: 无阵营，Boss战难度+10%(接入scaleEnemy)
  - `altruist/ego/balance`: 最终战NPC助战/属性加成比例
- **触发时机**: showStory()开头 + showChapterIntro()末尾 + completeStoryChapter()完成后
- **选择回顾**: showStory()底部显示已做选择

### 2. 连锁任务系统 (QUEST_CHAINS)
- **6条任务链**: 白城三杰/铁岭锻造之路/潮汐之谜/雾中幽魂/龙之宿命/深渊真言
- **8个链内新任务**(CHAIN_QUESTS): sq_white_guardian/sq_iron_vein/sq_masterwork/sq_sea_secret/sq_kraken_horn/sq_spirit_pact/sq_ancient_grove/sq_dragon_blood
- **任务链完成奖励**: XP + 金币 + 永久属性加成
- **任务链进度UI**: 公会场景显示进度条和完成状态
- **自动解锁**: 完成链内一个任务自动解锁下一个

### 3. 系统集成
- `_findQuest()`: 扩展搜索范围(QUESTS + SIDE_QUESTS + CHAIN_QUESTS + guildQuests)
- `sceneGuild()`: allAvailable包含SIDE_QUESTS
- `advanceSideQuest()`: 完成时调用checkQuestChainCompletion
- `completeQuest()`: 完成时调用checkQuestChainCompletion
- `generateRandomDrop()`: _lootBonus选择影响掉落
- `scaleEnemy()`: _neutralPath选择影响Boss难度
- `getNpcDiscount()`: _humanAlly选择影响铁匠折扣
- `addNpcAffection()`: _socialBonus选择影响好感度

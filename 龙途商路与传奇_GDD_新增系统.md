# 《龙途：商路与传奇》GDD更新 — 六大新增系统

---

## 一、幸运属性系统

### 1.1 设计目的
打造"隐性上头引擎"——幸运不影响常规战斗平衡，只影响概率性事件，让玩家为了"下一次好运"反复尝试。

### 1.2 幸运值获取
- 初始幸运：5点（所有角色一致）
- 升级成长：每级 +0.5 幸运（向下取整，50级满级时基础幸运约30）
- 装备加成：饰品、腰带可提供幸运词条
- 特殊道具：幸运草、命运骰子等消耗品临时增加幸运

### 1.3 幸运值影响

**会心一击（致命一击）：**
- 触发率 = 幸运值 × 0.5%（幸运30时 = 15%触发率）
- 效果：造成 250% 伤害（暴击是 150%）
- 文字提示："⚡致命一击！你的剑刃精准刺穿了敌人的护甲！"
- 和暴击独立计算，可叠加（先判定会心，再判定暴击，双重触发时 375% 伤害）

**装备掉落品质：**
- 幸运值每 10 点提升一档掉落品质权重
- 幸运 0-9：普通 70% / 精良 25% / 稀有 5%
- 幸运 10-19：普通 50% / 精良 35% / 稀有 13% / 史诗 2%
- 幸运 20-29：普通 30% / 精良 40% / 稀有 22% / 史诗 7% / 传说 1%
- 幸运 30+：普通 15% / 精良 35% / 稀有 32% / 史诗 15% / 传说 3%

**装备词条品质：**
- 幸运值影响随机词条的数值浮动
- 幸运 0-9：词条数值在 60%-100% 范围内随机
- 幸运 10-19：词条数值在 70%-110% 范围内随机
- 幸运 20-29：词条数值在 80%-120% 范围内随机
- 幸运 30+：词条数值在 90%-130% 范围内随机

**赌博系统：**
- 赌博类型：骰子猜大小、卡牌抽花色、转盘抽奖
- 基础胜率 30% + 幸运值 × 1%（上限 70%）
- 赌博奖励：金币、稀有道具、传说装备碎片
- 赌博消耗：金币或钻石，每日限 3 次免费

**其他影响：**
- 逃跑成功率：基础 50% + 幸运值 × 1%
- 宝箱陷阱触发率：基础 30% - 幸运值 × 0.5%
- 商路随机事件收益：幸运值 × 2% 额外金币加成

### 1.4 数值平衡目标
- 幸运值不直接增加战斗力，只影响概率
- 零氪玩家通过肝也能堆高幸运（装备+道具）
- 月卡玩家可购买幸运道具，但不卖幸运属性点

## 二、NPC好感度系统

### 2.1 设计目的
给玩家"社交养成"的钩子，和商贸系统联动，让交易不只是冷冰冰的买卖。

### 2.2 好感度等级

| 等级 | 名称 | 所需好感 | 解锁权益 |
|------|------|---------|---------|
| 1级 | 陌生 | 0 | 基础商品，无折扣 |
| 2级 | 熟悉 | 50 | 95折，解锁部分稀有商品 |
| 3级 | 信赖 | 150 | 90折，解锁专属任务线，可赠送礼物 |
| 4级 | 挚友 | 300 | 85折，解锁传说商品，赠送专属装备 |

### 2.3 好感度提升方式

**交易提升：**
- 每交易 100 金币 +1 好感
- 购买稀有商品 +3 好感
- 购买传说商品 +10 好感

**送礼提升：**
- 普通礼物（鲜花、酒、书籍）：+5 好感
- 稀有礼物（宝石、名画、古董）：+15 好感
- 专属礼物（NPC喜好物品）：+30 好感
- 每日送礼上限：3 次

**支线任务：**
- 完成NPC相关支线：+20~50 好感
- 剧情选择帮助NPC：+10 好感

**对话互动：**
- 每日首次对话：+1 好感
- 正确回应NPC话题：+2~5 好感

### 2.4 重要NPC设计

**每个城镇有 3-5 个重要NPC：**

| NPC类型 | 功能 | 挚友解锁权益 |
|--------|------|-------------|
| 铁匠 | 装备购买/修理/强化 | 解锁传说武器锻造，修理免费 |
| 商人 | 货物交易/市场情报 | 解锁黑市商品，独家折扣 |
| 药剂师 | 药水购买/配方学习 | 解锁稀有药水配方，免费赠送药水 |
| 旅店老板 | 休息恢复/任务接取 | 免费住宿，独家任务线 |
| 神秘商人 | 稀有道具/赌博 | 解锁传说装备抽奖，赌博胜率+5% |

### 2.5 好感度衰减（可选）
- 超过 7 天不互动，每日 -1 好感（最低降至信赖等级）
- 目的：鼓励玩家定期维护NPC关系

## 三、装备随机属性系统

### 3.1 设计目的
打造核心"刷装备"上头引擎，参考暗黑破坏神的装备系统。

### 3.2 装备品质阶梯

| 品质 | 颜色 | 基础属性倍率 | 词条数量 | 前缀数量 | 掉落权重（基础） |
|------|------|------------|---------|---------|----------------|
| 普通 | 白色 | 100% | 0-1 | 0 | 60% |
| 精良 | 绿色 | 110% | 1-2 | 1 | 25% |
| 稀有 | 蓝色 | 125% | 2-3 | 1-2 | 10% |
| 史诗 | 紫色 | 145% | 3-4 | 2 | 4% |
| 传说 | 橙色 | 170% | 4-5 | 2-3 | 0.9% |
| 神话 | 红色 | 200% | 5-6 | 3 | 0.1% |

### 3.3 装备前缀系统

**属性前缀（1个）：**
- 狂暴的：+15% 攻击力
- 坚韧的：+20% 防御力
- 狡诈的：+10% 闪避率
- 智慧的：+15% 魔法伤害
- 幸运的：+5~15 幸运值
- 迅捷的：+10% 攻击速度
- 巨人的：+25% 生命值上限

**效果前缀（1-2个）：**
- 燃烧之：攻击附带火焰伤害
- 冰霜之：攻击附带减速效果
- 吸血之：攻击恢复生命值
- 暴击之：+10% 暴击率
- 会心之：+5% 会心一击率
- 反击之：受到攻击时 20% 概率反击
- 连击之：攻击时 10% 概率追加一次攻击

**前缀组合示例：**
```
【传说·长剑】"狂暴的燃烧之会心长剑"
攻击力：85-102（传说品质倍率）
词条：
  +15% 攻击力（狂暴前缀）
  攻击附带 15 火焰伤害（燃烧之效果前缀）
  +5% 会心一击率（会心之效果前缀）
  +12% 暴击率（随机词条）
  +35 力量（随机词条）
  攻击时有 5% 概率造成双倍伤害（随机词条）
```

### 3.4 套装系统

**套装设计原则：**
- 每个套装 6 件装备，玩家可收集 2/4/6 件分档奖励
- 套装名取自 D&D 经典神器或传说
- 套装有主题性（输出型/防御型/辅助型）

**示例套装：**

**【火焰领主套装】（输出型）：**
- 2件：+15% 火焰伤害
- 4件：攻击时有 20% 概率触发火焰爆发（对周围敌人造成 50 火焰伤害）
- 6件：免疫火焰伤害，火焰技能冷却 -30%

**【守护者套装】（防御型）：**
- 2件：+20% 防御力
- 4件：受到致命伤害时，有 50% 概率保留 1 点生命（冷却 120 秒）
- 6件：每 5 秒恢复 5% 最大生命值

**【幸运商人套装】（辅助型）：**
- 2件：+10 幸运值
- 4件：商店购买价格 -10%，出售价格 +10%
- 6件：掉落品质提升一档（幸运 30 时史诗概率翻倍）

### 3.5 装备掉落来源

| 来源 | 普通概率 | 史诗+概率 | 传说概率 |
|------|---------|----------|---------|
| 普通怪物 | 15% | 0.5% | 0.01% |
| 精英怪物 | 50% | 5% | 0.3% |
| Boss | 100% | 25% | 3% |
| 迷宫宝箱 | 80% | 15% | 2% |
| 迷宫最终宝箱 | 100% | 40% | 8% |
| NPC挚友赠送 | 100% | 50% | 10% |

## 四、装备槽位扩展

### 4.1 七槽位设计

| 槽位 | 主要属性倾向 | 次要属性倾向 | 特殊效果倾向 |
|------|------------|------------|------------|
| 武器 | 攻击力、力量 | 暴击、命中 | 技能加成、元素伤害 |
| 头盔 | 防御力、智力 | 魔法值、抗性 | MP恢复、技能冷却 |
| 铠甲 | 防御力、体力 | 生命值、抗性 | 伤害减免、反伤 |
| 手套 | 攻击速度、敏捷 | 暴击、命中 | 连击概率、抓取特效 |
| 靴子 | 闪避率、敏捷 | 移动速度、体力 | 逃跑成功率、陷阱规避 |
| 腰带 | 体力、负重 | 生命值、防御 | 格挡率、背包扩展 |
| 饰品 | 幸运、魅力 | 全属性 | 特殊技能、套装核心件 |

### 4.2 属性分配示例

**武器（长剑）：**
```
基础属性：攻击力 50-65（品质倍率浮动）
词条倾向：力量 +暴击 +命中 火焰伤害 攻击速度
前缀倾向：狂暴的 燃烧之 暴击之 会心之
```

**靴子（皮靴）：**
```
基础属性：闪避率 +3%
词条倾向：敏捷 +体力 +移动速度 +闪避率 逃跑成功率
前缀倾向：迅捷的 狡诈的 冰霜之 反击之
```

**腰带（力量腰带）：**
```
基础属性：负重 +20，体力 +15
词条倾向：力量 +体力 +生命上限 +格挡率 幸运值
前缀倾向：巨人的 坚韧的 幸运的 反击之
```

### 4.3 装备重量与负重

- 每件装备有重量值（武器重、饰品轻）
- 玩家负重上限 = 基础 100 + 力量 × 2 + 腰带加成
- 超负重效果：移动速度 -20%，闪避率 -10%
- 目的：给玩家配装策略空间（重甲高防但重，轻甲灵活但脆）

## 五、随机迷宫系统

### 5.1 设计目的
打造"无限内容"，解决内容消耗过快问题，给玩家持续刷装备的场所。

### 5.2 迷宫结构

**迷宫生成规则：**
- 每个迷宫有固定入口（城镇→迷宫传送点）
- 迷宫深度：5/10/15/20 层（难度递增）
- 每层结构：1-3 个十字路口节点 + 随机房间
- 迷宫每次进入重新随机生成（Roguelike）

**节点类型：**
- 十字路口：4 个方向选择（已探索/未探索标记）
- 战斗房间：遭遇 1-3 波怪物
- 宝箱房间：随机品质宝箱
- 商人房间：神秘商人（价格略高）
- 陷阱房间：触发陷阱（伤害/减益）
- 空房间：无事发生，可休息恢复
- Boss房间：迷宫最深层（必出史诗+装备）

### 5.3 简单可视化设计

**路口显示：**
```
【迷宫·第3层】
你站在一个十字路口，火把的光芒摇曳。

    [?] 上方通道未知
[✓] 左侧已探索      右侧未探索 [?]
    [↓] 返回上一层

【地图】(已探索: 4/12)
  [●]→[?] 
   ↓
  [●]→[✓]→[?] 
   ↓
  [你]

请选择方向：
1. 向上探索
2. 向右探索
3. 向下返回
4. 查看详细地图
```

**房间显示：**
```
【房间·战斗】
你推开门，发现 3 只骷髅战士正盯着你！

骷髅战士A（等级 12）：HP ??? / ???
骷髅战士B（等级 12）：HP ??? / ???
骷髅战士C（等级 13）：HP ??? / ???

请选择行动：
1. ⚔️ 战斗
2. 🏃 逃跑（成功率 65%）
3. 🔍 侦查（消耗 5 MP，显示敌人详细信息）
```

### 5.4 迷宫类型

| 迷宫名称 | 深度 | 难度 | 特色 | 推荐等级 |
|---------|------|------|------|---------|
| 哥布林巢穴 | 5层 | 简单 | 高金币掉落，适合新手 | 5-15 |
| 幽暗矿洞 | 10层 | 中等 | 高矿石掉落，可采集材料 | 15-25 |
| 亡者地宫 | 15层 | 困难 | 高装备掉落，亡灵系怪物 | 25-35 |
| 龙之墓穴 | 20层 | 极难 | 传说装备概率最高，Boss必出史诗 | 35-50 |

### 5.5 迷宫奖励

**首刷奖励：**
- 经验值：普通怪物 50 EXP × 迷宫层数 × 首刷倍率（3倍）
- Boss：500 EXP × 迷宫层数 × 首刷倍率（3倍）
- 宝箱：基础品质 +1 档

**后续刷取：**
- 经验值：普通怪物 50 EXP × 迷宫层数 × 0.3倍
- Boss：500 EXP × 迷宫层数 × 0.3倍
- 装备/宝物掉落：概率不变（驱动玩家为装备而刷）

**每日首刷标记：**
- 每个迷宫每日首次通关享受"首刷倍率"（1.5倍经验）
- 鼓励玩家每日尝试不同迷宫

### 5.6 迷宫消耗与限制

- 进入迷宫消耗：体力 10 点（上限 100，每小时恢复 10 点）
- 月卡玩家：体力上限 +50，恢复速度 ×1.5
- 每日进入次数：普通玩家 3 次，月卡玩家 5 次

## 六、首刷高经验/后续低经验机制

### 6.1 设计目的
- 防止玩家在一个迷宫刷到满级（破坏游戏节奏）
- 驱动玩家为装备而刷，不是为经验而刷
- 鼓励玩家探索新迷宫

### 6.2 经验倍率规则

| 刷取次数 | 经验倍率 | 掉落倍率 | 说明 |
|---------|---------|---------|------|
| 第1次（首刷） | 300% | 100% | 推动剧情，高经验 |
| 第2次 | 50% | 100% | 经验骤降，掉落不变 |
| 第3次+ | 30% | 100% | 持续低经验，掉落不变 |
| 每日重置 | 首刷倍率恢复为150% | 100% | 鼓励每日首刷 |

### 6.3 玩家心理曲线

```
首刷高经验 → 快速升级 → 解锁新内容 → 发现装备不够强 → 回头刷装备
                                          ↓
                              后续刷取经验低 → 不想刷 → 退出游戏（问题）
                                          ↓
                              解决：装备掉落不变 → "再刷一次，说不定出传说" → 持续留存
```

### 6.4 平衡调参

**经验值设计目标：**
- 首刷通关 20 层迷宫：约 5000 EXP（相当于 2-3 级）
- 后续刷取：约 500 EXP（几乎无升级感）
- 传说装备掉落率：迷宫 Boss 约 8%（12.5 次 Boss 战出一件）

**上头循环：**
```
刷迷宫 → Boss战 → 出史诗装备（开心）→ 但不是传说 → "再刷一次"
         ↓
       出传说装备（狂喜）→ 换装 → 发现属性不是最优 → "再刷一件更好的"
```

## 七、系统联动总结

### 7.1 六大系统如何协同

```
【刷装备循环】
随机迷宫 → 战斗 → 装备掉落 → 幸运值影响品质 → 获得极品装备 → 好感度解锁商店 → 购买礼物 → 提升NPC好感 → 解锁传说商品 → 强化装备 → 挑战更高难度迷宫

【首刷推动剧情】
新迷宫首刷 → 高经验 → 快速升级 → 解锁新城镇 → 新NPC → 新商路 → 新装备槽位 → 新套装收集目标

【上头核心】
"下一次会心一击" + "下一个极品掉落" + "下一件传说装备" + "下一个NPC挚友"
```

### 7.2 修改后的游戏支柱

**原三大支柱：**
1. 选择即命运
2. 双循环共振
3. 简单上头

**新增第四支柱：**
4. **刷出传奇** —— 装备系统成为核心上头引擎
5. **社交共谋** —— 全服隐藏任务驱动社交裂变和社群活跃
6. **赛季重生** —— 新区域+赛季词条=永远有新目标

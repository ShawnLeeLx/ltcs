# 2026-06-05 开发日志

## V23 好感度系统 90%→100%

### 缺失函数补全（4个"幽灵调用"——函数名在代码里但未定义）
- `getNpcAffection(npcId)`: 从G.npcRelations读取好感度值
- `getNpcAffectionTier(npcId)`: 根据好感度返回tier对象（含title/emoji/color），使用NPC_AFFECTION_TIERS常量
- `addNpcAffection(npcId, amount)`: 增加好感度+自动检测40/70/100阈值触发解锁
- `giveGiftToNpc(npcId, materialId)`: 送礼逻辑——消耗材料、计算好感（喜爱2倍）、触发解锁
- `getNpcDiscount(locId)`: 从G._npcDiscounts读取折扣（上限50%）

### 解锁兑现系统 `_applyNpcUnlock`
- discount: 存入G._npcDiscounts，getNpcDiscount自动应用
- buff: 存入G._npcBuffs，战斗属性函数自动加成（getEffStat/calcMaxHp/calcAtk/calcDef）
- skill: 自动push到G.skills + 存入G._npcSkills
- recipe: 存入G._unlockedRecipes
- item: 存入G._pendingNpcItems队列

### 15个NPC好感技能加入SKILLS数组
guardian_stance, dragon_tongue, fate_eye, mist_walk, poison_clear, nature_shield, arcane_sight, storm_step, dragon_breath, iron_formation, frost_armor_skill, dragon_speech, ash_walk, flame_sense, abyss_eye, shadow_stealth

### 送礼入口补全
- sceneShop: 新增🎁送礼按钮 + showGiftPanel的shop场景NPC映射
- sceneGuild: 新增🎁送礼按钮 + showGiftPanel的guild场景NPC映射

### 数据格式修复
- sceneNpcRelations: NPC_DEFS是对象不是数组，用for-in遍历，用npc.loc替代npc.location
- _renderNpcRelationCard: tier是对象，取tier.title/tier.emoji/tier.color；likes显示中文名
- completeQuest: NPC_DEFS查找从.find改为for-in遍历

### 战斗属性集成
- getEffStat(stat): 末尾累加G._npcBuffs中对应stat的buff值
- calcMaxHp: 加_npcBuffs中hp buff
- calcAtk: 加_npcBuffs中atk buff
- calcDef: 加_npcBuffs中def buff

### 最终状态
- 龙途_手机试玩版_V23.html：13551行/591KB/JS语法通过/0重复函数

## V23 战斗系统增强 80%→85%

- 深渊战斗初始化：boss/elite敌人实例添加skills字段拷贝

## 立绘/场景图方案讨论

用户询问手绘素描写实风格的任务立绘（主角/NPC/敌人）+ 世界地图/城镇场景的免费解决方案。

### 推荐方案
1. **本地SD**：Draw Things（Mac M芯片原生App）+ Realistic Vision模型 + sketch LoRA
2. **在线免费**：Civitai / Tensor.art / Leonardo.ai（每日免费额度）
3. **风格一致性**：同Seed + 固定提示词前缀 / 训练轻量LoRA（20张样本）
4. **素材库**：OpenGameArt / itch.io 搜 hand-drawn RPG assets

### 提示词模板方向
- 角色：`pencil sketch portrait, [description], eastern fantasy, detailed ink drawing, white background, RPG character sheet`
- 敌人：`pencil sketch creature, [monster type], dark atmosphere, detailed line art, RPG bestiary illustration`
- 场景：`pencil sketch landscape, [location], eastern fantasy village, detailed architectural drawing`
- 输出：WebP压缩50-100KB/张，base64内嵌HTML

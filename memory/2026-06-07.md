# 2026-06-07 开发日志

## 坠星湖无法进入Bug修复（00:30-00:44）

### 根因
`travelTo()` 函数中 `var loc = LOCATIONS[locId]` 声明在函数末尾，但 `loc.type` 在中间被引用（10%遇敌判断）。由于 `var` 提升，`loc` 为 `undefined`，访问 `undefined.type` 抛 TypeError，导致函数在 `G.location = locId` 之后崩溃。

### 修复内容
1. `travelTo`: 将 `var loc` 移到函数开头
2. `renderMiniMap`: `cellData.type` → `room.terrain`
3. `sceneWorldMap`: 非城镇+有miniMap自动跳转
4. `travelTo`: 已在地点时直接进入场景

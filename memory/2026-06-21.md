## V26 仙剑式战斗菜单改造（22:00完成推送）

commit 8566797，用户5个问题触发战斗系统改造。

### 用户问题回答
1. ⚔️⚡ = 快速攻击（自动选敌人），已移除合并进R重复操作
2. 观察四周 = 确实冗余，已改为🔄刷新
3. 随机事件按钮 = 和地块🔍emoji是同一系统（radar点击跳转），不是两套独立系统
4. 首次战斗引导 = 已添加，turn=1时自动弹出说明
5. 仙剑式战斗 = 已改造

### 战斗按钮改为5+2布局（参考仙剑1）
| 按钮 | 对应仙剑 | 说明 |
|------|----------|------|
| ⚔️ 攻 | 攻击 | 普通攻击，选目标 |
| 📖 术 ▶ | 仙术 | 展开法术+技能+天赋统一列表（原来3个独立展开按钮合并） |
| 🎒 物 ▶ | 物品 | 展开背包可用品（药水/食物，最多6件），替代原来的全功能背包入口 |
| 🛡️ 防 | 防御 | 本回合减伤 |
| 🏃 逃 | 逃跑 | 逃跑 |
| R ⚔️ | R键 | 重复上回合操作（攻击/防御/技能/天赋） |
| ▶/⏹ | A键 | 自动战斗开关 |

### 移除的按钮
- ⚔️⚡快速攻击 → 合并进R
- 🎒背包(全功能) → 改为物品(只显示可用品)
- 独立天赋技能按钮 → 合并进"术"

### 新增函数
- `repeatLastAction()`：根据G._lastBattleAction重复上回合操作
- `useBattleItem(idx)`：战斗中使用背包物品（自动减数量，0时移除）
- `_afterPlayerAction()`：玩家操作后通用处理（回合+1、冷却-1、触发敌回合）

### 追踪变量
- `G._lastBattleAction`：'attack'/'defend'/'skill'/'origin'/'item'
- `G._lastSkillId`：上次使用的技能ID
- `G._battleTutorialDone`：是否已完成战斗引导
- `G._showSkillMenu`：术菜单展开状态
- `G._showBattleItems`：物菜单展开状态

### 首次战斗引导
turn=1且G._battleTutorialDone未设置时弹出对话框，说明各按钮作用，点击"开始战斗"关闭

# 天气系统实现 — 2026-06-11

## 目标
为龙途传说V24实现完整的天气系统，影响战斗、旅行和商贸。

## 实现内容

### 核心数据
- **8种天气类型**：晴朗☀️、多云⛅、雨天🌧️、暴风雨⛈️、大雾🌫️、降雪❄️、暴风雪🌨️、酷暑🔥
- **3个区域**：北方冻土(偏向寒冷)、中原沃土(温和均衡)、南方龙域(偏向炎热)
- **21个地点→区域映射**：WEATHER_LOC_REGION

### 天气引擎
- `tickWeather()` — 每5次场景切换触发天气变化
- 每区域独立天气，60%概率变化，40%保持不变
- `_initWeather()` — 延迟初始化，存档兼容

### 战斗效果
- **物攻修正**：暴风雨-10%、降雪-8%、暴风雪-15%、雨天-5%、酷暑-5%
- **防御修正**：暴风雪+8%、降雪+5%、大雾+3%、暴风雨-8%、酷暑-5%
- **魔攻修正**：暴风雨+5%、酷暑+5%、暴风雪-10%、大雾-5%
- **闪避修正**：暴风雪-12%、暴风雨-10%、雨天-5%、降雪-5%、大雾+5%
- **暴击修正**：暴风雨+5%、暴风雪+5%、酷暑+3%
- 集成点：`playerAttack`（物攻+暴击）、`calcSkillDmg`（技能伤害）、`enemyTurn`（防御+闪避）

### 旅行效果
- 天气影响遇敌率：暴风雪+20%、暴风雨+15%、大雾+8%、雨天+5%
- 旅行叙述附加天气描述：`w.emoji + w.travelMsg`

### 商贸效果
- 天气影响交易价格：暴风雪+15%、暴风雨+10%、雨天+5%、降雪+8%、酷暑-5%
- 集成点：`getTradePrice`（weatherMod乘数）

### UI
- `renderWeatherBar(locId)` — 城镇/战斗中的天气状态条
- `sceneWeatherForecast()` — 三区域天气预报面板（含效果详情+已解锁地点）
- 城镇按钮栏新增🌤️天气按钮

### 成就（3个）
- 🌤️ 观天者：查看天气预报5次 → 100金
- ⛈️ 暴风猎人：暴风雨下赢3场战斗 → 500金
- 🌨️ 暴风雪幸存者：暴风雪下赢1场战斗 → 800金

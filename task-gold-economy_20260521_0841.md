# 金币经济重设计完成

## 目标
怪不再掉金币，让金币有独特价值——通过商贸/卖物/任务获得，通过打造/训练/旅店消耗。

## 已完成改动

### 1. 战斗不再掉金币
- endBattle(): 移除goldMin/goldMax，改为材料掉落(ENEMY_MAT_DROPS表)
- renderBattle内联胜利路径: 同步移除金币，加材料掉落
- 宝箱(miniMapTreasure): 改为掉1-2普通材料+20%稀有材料

### 2. 铁匠铺打造系统 (🔨 tab)
- 11个配方: 普通(3) → 优秀(3) → 精良(2,可选词条) → 史诗(2,可选词条) → 传说(1)
- 材料+金币→定制装备
- 稀有以上配方可选1个属性词条(力量/敏捷/智力/魅力/幸运/攻击/防御/HP/暴击/闪避/吸血)
- 缺材料配方灰显，材料背包摘要显示

### 3. 训练场 (🏋️)
- 花金币获得属性点: STR/AGI/INT/CHA/LUK 各+1
- 费用递增: 50 + 已训次数×20 金
- 每日次数限制: floor(level/10)*5+5
- 20级以上可"集中特训"(5次)，获得5自由属性点
- 旅店休息重置每日训练次数

### 4. 传送卷轴
- 潮汐港独家商品 50金
- 使用后选已解锁城镇传送

### 5. sellItem增强
- 支持材料sellPrice字段

## 金币流转验证
- ✅ 战斗掉落: 0金(只掉材料+装备)
- ✅ 金币来源: 商贸>卖物>章节奖励>任务奖励
- ✅ 金币消耗: 打造>训练>旅店>商店>传送

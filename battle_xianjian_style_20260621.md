# V26 仙剑式战斗菜单改造 - 2026-06-21

## 目标
参考仙剑奇侠传1战斗系统，将混乱的战斗按钮改为清晰的攻/术/物/防/逃五按钮，加入首次战斗引导。

## 仙剑1战斗核心
- 回合制，每回合选一个动作
- 主菜单：攻击/仙术/物品/防御/逃跑/合击
- R键重复上回合操作，F键使用最强法术，A键自动攻击
- 每个按钮功能单一清晰，不需要猜

## 改造内容

### 战斗按钮改为5+2布局
| 按钮 | 作用 | 对应仙剑 |
|------|------|----------|
| ⚔️ 攻 | 普通攻击，选择敌人目标 | 攻击 |
| 📖 术 ▶ | 展开法术+技能+天赋列表 | 仙术 |
| 🎒 物 ▶ | 展开背包可用品(药水/食物，最多显示6件) | 物品 |
| 🛡️ 防 | 本回合防御，伤害减半 | 防御 |
| 🏃 逃 | 逃跑 | 逃跑 |
| R ⚔️ | 重复上回合操作(攻击/防御/技能/天赋) | R键 |
| ▶/⏹ | 自动战斗开关 | A键 |

### 移除的按钮
- ⚔️⚡快速攻击 → 合并进R重复操作
- 🎒背包(全功能) → 改为物品(只显示可用品)
- 独立天赋技能按钮 → 合并进"术"展开菜单

### 首次战斗引导
第一场战斗turn=1时弹出对话框：
- ⚔️ 攻：选择敌人进行普通攻击
- 📖 术：展开法术和技能列表
- 🎒 物：使用背包中的药水和物品
- 🛡️ 防：本回合防御，受到伤害减半
- 🏃 逃：尝试逃跑
- R：重复上回合操作
- ▶：自动战斗

### 新增函数
- `repeatLastAction()`：根据G._lastBattleAction重复上回合操作
- `useBattleItem(idx)`：战斗中使用背包物品
- `_afterPlayerAction()`：玩家操作后的通用处理(回合+1、冷却-1、触发敌回合)

### 追踪变量
- `G._lastBattleAction`：'attack'/'defend'/'skill'/'origin'/'item'
- `G._lastSkillId`：上次使用的技能ID
- `G._battleTutorialDone`：是否已完成战斗引导
- `G._showSkillMenu`：术菜单展开状态
- `G._showBattleItems`：物菜单展开状态

## 回答用户问题

3. 随机事件按钮 = 和地块🔍emoji是同一系统（radar点击跳转），不是两套独立系统
4. 首次战斗引导 = ✅ 已添加，turn=1时自动弹出
5. 仙剑式战斗 = ✅ 已改造为攻/术/物/防/逃+R重复

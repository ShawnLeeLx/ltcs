## 一、等级上限扩展至100级

### 1.1 阶梯式属性点衰减

**核心问题：** 100级 × 3点/级 = 300点，5维各60上限 = 300 → build多样性消失。

**解决方案：**

| 等级段 | 每级属性点 | 等级跨度 | 小计 | 设计意图 |
|--------|-----------|---------|------|---------|
| 1-50 | 3点 | 50级 | 150点 | 快速成型期，build基本定调 |
| 51-80 | 2点 | 30级 | 60点 | 精修期，补短板或强化特色 |
| 81-100 | 1点 | 20级 | 20点 | 完美期，每点都珍贵 |

**升级总属性点：230点 + 初始50点 = 280点**
**5维各60上限 = 300点** → 100级也填不满，build选择始终有意义。

### 1.2 等级节奏设计

| 等级段 | 游戏阶段 | 主线对应 | 预计耗时 | 核心体验 |
|--------|---------|---------|---------|---------|
| 1-30 | 新手成长期 | 第1-3章 | 20小时 | 快速变强，爽感驱动 |
| 31-50 | 核心体验期 | 第4-6章 | 40小时 | build成型，策略展开 |
| 51-70 | 深度探索期 | 第7-8章+新区域 | 60小时 | 精修属性，挑战高难度 |
| 71-90 | 满级冲刺期 | 新区域+终局迷宫 | 80小时 | 装备毕业，深潜挑战 |
| 91-100 | 传奇期 | 终极挑战 | 不设上限 | 每级珍贵，社交角色转换 |

### 1.3 为什么不是200级

- 文字游戏升级感知弱，100级后边际快感递减太快
- 200级需要4-5倍内容量，小团队不现实
- 100级+阶梯衰减+赛季扩展 > 200级线性堆叠

## 二、赛季制新区域开放

### 2.1 赛季规划

**赛季1：「龙途初开」（上线）**
- 12个基础城镇 + 4个基础迷宫
- 等级上限 50
- 基础装备体系（10套套装）
- 基础商贸路线 20条

**赛季2：「海妖之歌」（上线后3个月）**
- 解锁群岛区域（6个新城镇 + 海战迷宫）
- 等级上限提升至 70
- 新装备体系（海妖套装、船员装备）
- 新机制：船队系统（海上贸易+海怪战斗）
- 新商贸路线：跨海贸易（高利润高风险）

**赛季3：「天空之城」（上线后6个月）**
- 解锁浮空岛区域（4个新城镇 + 空中迷宫）
- 等级上限提升至 85
- 新装备体系（风神套装、飞行坐骑）
- 新机制：空战系统
- 新商贸路线：空中贸易

**赛季4：「深渊裂隙」（上线后9个月）**
- 解锁地下世界（6个新区域 + 深渊迷宫）
- 等级上限提升至 100
- 新装备体系（深渊套装）
- 新机制：Roguelike深渊挑战
- 新商贸路线：地下黑市

### 2.2 赛季内容包构成

每个赛季包含：
| 内容 | 说明 |
|------|------|
| 新区域 | 4-6个新城镇/区域 |
| 新迷宫 | 2-3个新迷宫 |
| 新主线 | 2-3章新剧情 |
| 新NPC | 每个新城镇3-5个 |
| 新套装 | 3-5套新套装 |
| 新商路 | 5-10条新商贸路线 |
| 新隐藏任务 | 6个（每两周1个） |
| 新机制 | 1-2个新玩法系统 |
| 等级上限提升 | 15-20级 |

### 2.3 赛季与商业化联动

| 付费点 | 赛季联动 |
|--------|---------|
| Battle Pass | 每赛季1个，68元，含赛季专属外观 |
| 月卡 | 赛季首月1元体验，拉新回流 |
| 新区域礼包 | 30元，含过渡装备+商人引荐信 |
| 赛季隐藏装备 | 免费（靠隐藏任务获取） |
| 赛季外观 | 赛季限定，下赛季绝版 |

### 2.4 回流效应

```
新赛季上线 → 老玩家回流 → 带来新玩家 → DAU峰值
    ↓
赛季中期 → 内容消耗 → DAU回落（但高于上赛季均值）
    ↓
下赛季预告 → DAU再次回升 → 循环
```

## 三、终局系统：解决"无敌寂寞"

### 3.1 问题分析

**传统MMO的无敌寂寞 = 没人打过我 + 没新内容 → 无聊退坑**

**本游戏没有PVP**，问题简化为：
- 装备毕业 → 没东西刷
- 等级满了 → 没成长
- 内容通了 → 没新鲜感

### 3.2 解法A：深潜模式（动态难度）— P0，MVP就做

**核心：敌人数值随玩家成长，装备毕业=不存在的。**

**设计：**
- 通关普通迷宫后解锁"深潜模式"
- 深潜层数无上限，每层怪物数值+15%
- 掉落品质随深潜层数提升
- 深潜层数排行 = 终局竞争点

**深潜层数与掉落：**

| 深潜层数 | 怪物数值倍率 | 传说掉落率 | 蚀刻装备 |
|---------|------------|-----------|---------|
| 1-10 | 115%-260% | 基础率 | 无 |
| 11-30 | 275%-665% | 基础率×1.5 | 有（数值按层数缩放） |
| 31-50 | 680%-1640% | 基础率×2 | 蚀刻品质+1档 |
| 50+ | 1640%+ | 基础率×2.5 | 蚀刻品质+2档 |

**蚀刻装备：**
- 深潜模式专属掉落
- 基础属性随深潜层数缩放（层数越高数值越强）
- 有"蚀刻前缀"：深渊之（额外+10%伤害但-5%防御）、虚空之（+15%暴击但-10%生命）
- 蚀刻装备是真正的"无上限成长"系统

**深潜排行：**
- 全服深潜层数排行榜
- 每周结算，前10名获得"深潜者"称号+钻石奖励
- 排行榜驱动社交比较（不PK也能竞争）

### 3.3 解法B：赛季词条重置 — P2，赛季2开始

**核心：每个赛季的装备有"赛季专属词条"，赛季结束后失效。**

**赛季词条示例：**

| 赛季 | 赛季词条名 | 效果 |
|------|-----------|------|
| S1 龙途初开 | 龙裔亲和 | 对龙系怪物+15%伤害 |
| S2 海妖之歌 | 海风庇护 | 海上战斗+20%防御，海怪掉落+10% |
| S3 天空之城 | 风暴驾驭 | 空中战斗+25%闪避，飞行速度+15% |
| S4 深渊裂隙 | 深渊视界 | 深层迷宫可见范围+2，陷阱规避+20% |

**规则：**
- 赛季结束后，赛季词条变灰（不激活，但装备保留）
- 下赛季有新的赛季词条 → 需要重新刷当前赛季装备
- 老赛季装备的永久词条依然有效，只是赛季词条不再加成
- **这给了老玩家重新刷装的理由，同时不剥夺已有投入**

### 3.4 解法C：传奇社交角色 — P1，v1.0做

**核心：满级玩家从"冒险者"变成"传奇"，社交角色转换。**

**师徒系统：**
- 等级50+玩家可收徒（徒弟等级<30）
- 徒弟升级 → 师傅获得"师德点"
- 师德点兑换：专属外观、称号、NPC额外好感
- 徒弟出师（达到50级）→ 师傅获得"名师"成就
- 徒弟首充 → 师傅获得等值钻石（10%）

**传奇守护：**
- 帮助低等级玩家通关迷宫
- 守护者等级降低至副本匹配等级（不影响自身属性）
- 获得"守护者勋章" → 兑换专属物品
- 被守护的玩家通关率提升 → 双赢

**公会系统：**
- 等级50+才能创建公会
- 公会等级 = 成员活跃度总和
- 公会福利：集体商贸折扣、公会专属迷宫、公会仓库
- 公会排名 = 社交竞争点
- 公会战（非战斗）：商贸竞赛（谁更会赚钱）

**公会战设计（商贸竞赛）：**
```
赛季公会战：谁的商队最赚钱？

规则：
- 每周1次，持续24小时
- 公会成员商贸利润汇总
- 利润最高的公会获胜
- 奖励：公会专属称号+钻石+稀有商品优先购买权

不是打架，是比谁更会赚钱——这才是你游戏的调性。
```

### 3.5 终局系统优先级

| 优先级 | 系统 | 开发成本 | 效果 | 时间点 |
|--------|------|---------|------|--------|
| **P0** | 深潜模式 | 低（数值缩放公式） | 极高 | MVP |
| P1 | 师徒+传奇守护 | 中 | 高 | v1.0 |
| P1 | 公会系统 | 中 | 高 | v1.0 |
| P2 | 赛季词条重置 | 中 | 高 | 赛季2 |

## 五、更新后的游戏支柱

1. **选择即命运** —— 剧情分支+属性加点+build定义
2. **双循环共振** —— 战斗给商贸铺路，商贸给战斗供血
3. **简单上头** —— 界面简洁，决策有深度
4. **刷出传奇** —— 装备系统+深潜模式=无上限成长
5. **社交共谋** —— 隐藏任务+师徒+公会=社群活跃
6. **赛季重生** —— 新区域+赛季词条=永远有新目标

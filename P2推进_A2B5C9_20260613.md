## C9: 战斗数值平衡

### 问题诊断
- 原DIFF_SCALE=0.75，敌人固定不随等级增长
- 正常敌人：1回合秒杀（全等级无挑战）
- Boss：2-4回合击杀，剩余60-94% HP（过于简单）

### 修复方案
修改 `scaleEnemy()` 添加两层缩放：

```
lvScale = 1 + (playerLevel - 1) * 0.02    // 等级缩放
regScale = 2.0 (仅普通/精英敌人)           // 普通敌人额外缩放
effective = base * DIFF_SCALE * lvScale * regScale
```

### 设计理念
- 普通敌人：2回合击杀，持续消耗但无威胁（探索节奏）
- Boss：5-8回合击杀，需要策略和资源管理（紧张感）
- 最终Boss：理论上一对一不可胜，必须使用治疗/技能/道具（成就感）

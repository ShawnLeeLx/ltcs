## 遇到的问题
1. GitHub网页创建仓库时"couldn't check availability" + Create按钮无效 → 网络问题，最终成功创建
2. GitHub Pages API需要额外token权限 → 改为用户手动在Settings/Pages开启
3. gh CLI未安装 + Homebrew未安装 + 直接下载超时 → 改用git + token方式推送
4. 原仓库名longtu → 用户实际创建了ltcs

## 用户关注点
- 安全性：token权限范围、文件被篡改的影响
- 便利性：无限期token、链接不变自动更新

## 后续建议
- 用户删除已生成的deploy token（使用后即删）
- 更新游戏时重新生成token，推代码到main分支即可自动更新

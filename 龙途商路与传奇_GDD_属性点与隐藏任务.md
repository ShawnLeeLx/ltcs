# 《龙途：商路与传奇》GDD更新 — 属性点分配 & 全服隐藏任务

---

## 一、属性点自由分配系统

### 1.1 设计目的
让玩家定义自己的角色build，增加build多样性和重玩价值。从"系统分配属性"进化为"玩家选择属性"——这是D&D的核心体验。

### 1.2 等级与属性点

**等级上限：100级（阶梯式属性点衰减）**

**属性点获取：**
| 来源 | 属性点 | 说明 |
|------|--------|------|
| 初始属性 | 5维各10点（共50点） | 保证开局不偏科 |
| 升级1-50级 | 每级3点 | 150点，快速成型期 |
| 升级51-80级 | 每级2点 | 60点，精修期 |
| 升级81-100级 | 每级1点 | 20点，完美期 |
| 隐藏支线 | 约10个，每个+1~2点 | 约10-20点 |
| 世界事件 | 约5个，每个+1点 | 约5点 |
| 全收集成就 | 套装/图鉴/探索 | 约5点 |
| 赛季排名 | 每赛季+1~2点 | 持续运营 |
| **合计** | **约300-315点** | 含初始50点 |

**单属性上限：60点（含装备加成可超过）**
→ 5维上限共300点，升级获得约250点 → build选择始终有意义

赛季制新区域开放将逐步提升等级上限（50→70→85→100），详见《龙途商路与传奇_GDD_等级与终局.md》

### 1.3 五维属性效果

| 属性 | 每点效果 | 二次效果 | 对应build |
|------|---------|---------|----------|
| 力量 | +2 物理攻击 | +2 负重上限 | 狂战士/圣骑士 |
| 敏捷 | +1 攻击速度 +0.5% 闪避 | +0.3% 暴击率 | 刺客/弓手 |
| 智力 | +2 魔法攻击 +3 MP | +0.5% 技能冷却缩减 | 法师/术士 |
| 魅力 | -0.5% 商店价格 +0.3 NPC好感 | 解锁社交选项 | 商人/吟游诗人 |
| 幸运 | +0.5% 会心一击率 | 影响掉落品质/赌博/逃跑 | 赌徒/探险家 |

### 1.4 典型Build示例

**🗡️ 狂战士**（力量45/敏捷20/智力5/魅力5/幸运15）
- 特点：高物理伤害+高会心，几乎无魔法
- 弱点：无MP放技能，远程弱
- 适合：直接莽的玩家

**🧙 大法师**（力量5/敏捷10/智力45/魅力15/幸运15）
- 特点：魔法伤害爆炸，AOE清场
- 弱点：血量低，被近身就危险
- 适合：喜欢策略战斗的玩家

**💰 商人王**（力量10/敏捷10/智力15/魅力40/幸运25）
- 特点：商贸折扣+高幸运，越打越富
- 弱点：战斗力弱，需要靠装备弥补
- 适合：喜欢经营的玩家

**⚔️ 圣骑士**（力量25/敏捷10/智力20/魅力20/幸运5）
- 特点：均衡型，能打能奶能交易
- 弱点：没有突出优势
- 适合：什么都想体验的玩家

**🎲 赌徒**（力量10/敏捷15/智力10/魅力10/幸运45）
- 特点：极致幸运流，会心+赌博+掉落全拉满
- 弱点：常规战斗平庸，靠概率吃饭
- 适合：喜欢惊喜的玩家

**🏹 暗影刺客**（力量15/敏捷40/智力10/魅力5/幸运20）
- 特点：高闪避+高暴击+高会心，先手秒人
- 弱点：血少防低，一击不中被反杀
- 适合：喜欢刺激的玩家

### 1.5 特殊事件属性点来源

**隐藏支线（约10个，每个+1~2点）：**
- 被困的精灵：+1点（选择帮助）
- 古代竞技场冠军：+2点（连续战斗挑战）
- 知识之泉：+1点（正确回答3个世界谜题）
- 流浪的贤者：+2点（收集5本古籍交给他）
- ...

**世界事件（约5个，每个+1点）：**
- 龙潮来袭：全服击退龙群
- 商路劫难：护送商队安全到达
- 暗月之夜：在特定时间击败暗影怪物
- ...

**全收集成就：**
- 套装收集大师（集齐5套套装）：+1点
- 怪物图鉴完成50%：+1点
- 全城镇探索：+1点
- 全NPC达到信赖：+2点

**赛季排名奖励：**
- 赛季前10%：+2点
- 赛季前30%：+1点

### 1.6 洗点机制

| 次数 | 费用 | 说明 |
|------|------|------|
| 第1次 | 免费 | 新手容错 |

## 二、全服隐藏任务系统

### 2.1 设计目的
1. **DAU和留存的终极工具**——每两周一个新谜题=每两周一个回归理由
2. **社交裂变引擎**——线索靠NPC和社交传播，不是系统公告
3. **好感度价值放大器**——NPC从"打折工具"升级为"解谜必需品"

### 2.2 投放节奏

**每两周1个隐藏任务，每月2个**

**全年规划（24个隐藏任务）：**

| 月份 | 隐藏任务1 | 隐藏任务2 |
|------|----------|----------|
| 1月 | 消失的龙骑士 | 月光药剂师 |
| 2月 | 冰霜女王的遗物 | 熔炉之心 |
| 3月 | 暗影猎手之弓 | 海妖之歌 |
| 4月 | 春之祭典·花冠 | 荒野游侠的遗愿 |
| 5月 | 雷神之锤碎片 | 沙漠玫瑰之戒 |
| 6月 | 深渊守望者 | 夏至·太阳纹章 |
| 7月 | 风暴编织者 | 海盗王的罗盘 |
| 8月 | 炼金术士的秘典 | 幽灵船长的怀表 |
| 9月 | 秋收·丰收之角 | 星辰占卜师之眼 |
| 10月 | 万圣·南瓜王的冠冕 | 暗夜舞者的面纱 |
| 11月 | 铁匠的终极锻造 | 冬幕·冰晶之心 |
| 12月 | 年度终章·命运之轮 | 圣诞·极光斗篷 |

### 2.3 任务流程（3步设计）

```
第1步：获取线索
  → 与NPC对话，好感度越高线索越精确
  → 不同NPC给互补线索
  → 故意不做全服公告，靠社交传播

第2步：前往目标
  → 根据线索前往特定迷宫/区域
  → 完成特定战斗或找到特定物品
  → 可能需要特定好感度/属性值

第3步：获得奖励
  → 隐藏装备（独特效果，不超模）
  → 额外属性点（+1点）
  → 隐藏成就
```

### 2.4 NPC好感度与线索精度

**以"消失的龙骑士"为例：**

| NPC | 好感度 | 线索内容 |
|-----|--------|---------|
| 铁匠 | 陌生 | "最近没什么特别的。" |
| 铁匠 | 熟悉 | "听说北边废墟有异动，但我不确定。" |
| 铁匠 | 信赖 | "一个老冒险者说在幽暗矿洞深处看到了龙纹石碑，或许和龙骑士有关。" |
| 铁匠 | 挚友 | "听好了，龙骑士之剑被封印在幽暗矿洞第7层隐藏房间。你需要先在第3层找到龙纹钥匙才能打开封印门。我帮你标在地图上了。" |
| 旅店老板 | 信赖 | "最近有个神秘人在打听龙骑士的传说，他住在白城东边旧灯塔。" |
| 药剂师 | 熟悉 | "龙骑士？这里有本旧书提到过...但我不太信任陌生人。" |
| 神秘商人 | 挚友 | "龙骑士之剑的封印需要龙焰才能打开。我这有一瓶龙焰精华，但只卖给老朋友。1000金币，不还价。" |

**线索设计原则：**
- 至少2个NPC提供不同角度的线索（互补不是重复）
- 陌生级NPC几乎不给线索（激励培养好感度）
- 挚友级NPC给出关键信息+便利（地图标记、专属商品）
- 所有线索拼起来才能完成任务（不能只靠一个NPC）

### 2.5 隐藏装备设计

**核心原则：同等数值，独特效果，有趣但不必须**

**【龙骑士之剑】（武器·传说）**
```
攻击力：85-102
基础词条：+15%攻击力 +10%暴击率 +30力量
独特效果：对龙系怪物伤害+50%，击杀龙系怪物恢复10%生命值
独特前缀：龙焰之（攻击附带龙焰伤害，对非龙系也有15%额外伤害）
独特外观：剑身环绕龙焰特效
对比迷宫掉落传说武器：攻击力相同，暴击率略低，但龙系场景碾压
```

**【月光药壶】（饰品·传说）**
```
基础词条：+15智力 +20%药水效果 +10%技能冷却缩减
独特效果：每场战斗开始时自动使用一次随机药水效果
  （恢复30%生命/恢复30%MP/增加10%攻击/增加10%防御，随机一种）
独特外观：壶中月光流转
```

**【冰霜王冠】（头盔·传说）**
```
基础词条：+25防御力 +15智力 +20%冰霜抗性
独特效果：受到近战攻击时20%概率冻结攻击者1回合
独特外观：头顶冰晶光环
```

### 2.6 社交裂变机制

**不做全服公告的原因：**
- 有公告 → 所有人同时做 → 几小时内全服完成 → 无讨论价值
- 无公告 → 线索靠社交传播 → 持续数天的社群讨论 → DAU提升

**社交传播路径：**
```
第1天：少数高好感度玩家获得完整线索 → 前往目标 → 获得隐藏装备
       ↓
第2-3天：这些玩家在群聊/论坛分享线索 → 中等好感度玩家尝试
       ↓
第4-5天：线索扩散 → 大量玩家参与 → 讨论如何完成 → 社群活跃
       ↓
第6-7天：攻略出现 → 剩余玩家完成 → 期待下一个隐藏任务
```

**分享激励（不影响平衡）：**
- 分享线索到微信群：获得50金币奖励
- 帮助其他玩家完成隐藏任务：获得"引路人"称号
- 首个完成全服隐藏任务的玩家：获得"先驱者"称号+专属外观

### 2.7 与其他系统的联动

| 系统 | 联动方式 |
|------|---------|
| NPC好感度 | 核心联动：好感度=线索精度，隐藏任务最大价值放大器 |
| 迷宫系统 | 大部分隐藏任务目标在迷宫中，驱动玩家刷迷宫 |
| 幸运属性 | 幸运值高可能触发隐藏NPC对话选项 |
| 装备系统 | 隐藏装备有独特效果和外观，驱动收集欲 |
| 属性点 | 每个隐藏任务奖励+1属性点，养成价值 |
| 商贸系统 | 部分隐藏任务需要购买特定商品（如龙焰精华） |

### 2.8 运营节奏

```
第1周（投放周）：新隐藏任务上线
  → 无系统公告
  → 高好感度玩家开始发现线索
  → 社群开始讨论

第2周（扩散周）：线索逐渐扩散
  → 中等玩家开始参与
  → 攻略开始出现
  → 隐藏装备开始在排行榜出现

第3周（过渡周）：上个任务结束，新任务预告
  → 剩余玩家完成
  → 下个隐藏任务线索开始在NPC对话中出现
  → 玩家开始培养相关NPC好感度（预期驱动）
```

## 三、对"复杂度 vs 可玩性"的回答

### 隐藏任务增加可玩性，不增加系统复杂度

**原因：**

1. **不新增系统**——只是把NPC好感度+迷宫+装备三个已有系统重新组合，加一层叙事包装
2. **3步流程**——找线索→去地方→拿奖励，不需要新增解谜小游戏
3. **可选内容**——不完成隐藏任务不影响主线进度
4. **数值不超模**——隐藏装备和随机极品相当，独特但不碾压

**真正增加的"复杂度"是运营侧：**
- 每2周需要设计1个新隐藏任务（线索、NPC对话、目标地点、奖励装备）
- 需要维护NPC对话的时效性（隐藏任务期间特殊对话）

**但这个"运营复杂度"是值得的**——因为它直接驱动DAU、留存和社交裂变，是月收入2万+的关键杠杆。

### 属性点系统增加可玩性，略微增加数值平衡复杂度

**新增的平衡工作：**
- 需要确保极端build（如全幸运）不会破坏游戏体验
- 需要设计build验证内容（某些迷宫/事件对特定build更友好）
- 洗点机制需要平衡（不能太便宜否则build无意义，不能太贵否则新手被惩罚）

**但这些都是常规RPG数值工作**，不构成额外风险。

## 四、更新后的游戏支柱

1. **选择即命运** —— 剧情分支+属性加点+build定义
2. **双循环共振** —— 战斗给商贸铺路，商贸给战斗供血
3. **简单上头** —— 界面简洁，决策有深度
4. **刷出传奇** —— 装备系统成为核心上头引擎
5. **社交共谋** —— 隐藏任务驱动社交裂变和社群活跃

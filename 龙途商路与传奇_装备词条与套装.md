## 一、词条库（120+词条）

### 1.1 词条分类体系

词条分为5大类：
- **基础属性词条**：直接加属性值（力量+15、生命+50等）
- **百分比词条**：按百分比提升（攻击力+10%、暴击率+5%等）
- **效果词条**：触发特殊效果（攻击附带火焰、击杀恢复生命等）
- **条件词条**：满足条件时生效（对龙系+20%伤害、生命低于30%时+15%攻击等）
- **技能词条**：影响特定技能（火球术伤害+20%、技能冷却-15%等）

## 三、10个完整套装设计

### 3.1 套装设计原则

- 每套6件，分2/4/6件档位奖励
- 2件：基础加成（容易达成）
- 4件：核心效果（主要目标）
- 6件：终极形态（完美主义者）
- 套装名取自D&D经典或原创传说

### 3.2 输出型套装（4套）

## 四、装备生成规则

### 4.1 词条生成流程

```
1. 确定品质 → 决定词条数量范围
2. 随机词条类型 → 按槽位倾向加权
3. 随机词条数值 → 在范围内随机（受幸运值影响）
4. 生成前缀 → 按品质决定前缀数量
5. 检查套装归属 → 5%概率为套装件
6. 最终命名 → 前缀+装备名
```

### 4.2 词条数值浮动

基础浮动范围 × 幸运修正：
- 幸运 0-9：60%-100%
- 幸运 10-19：70%-110%
- 幸运 20-29：80%-120%
- 幸运 30+：90%-130%

### 4.3 套装件掉落规则

- 普通怪物：套装件概率 0.5%
- 精英怪物：套装件概率 2%
- Boss：套装件概率 8%
- 迷宫最终宝箱：套装件概率 15%
- 同一套装的不同部位掉落概率均等
